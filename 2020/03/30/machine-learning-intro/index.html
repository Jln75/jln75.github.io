Introduction

Let’s imagine we take a cell from a human body. This cell has many charactéristics.

radius mean texture mean perimeter mean area mean melignant
17,99 10,38 122,8 1001 ?

A malignat tumor is a tumor that spread to the surronding tissues, or around the body and can lead to death. The ability to early detect malignat cells, increases a patient’s chances of survival.

Most of the people may think that only a doctor with years of practicing could be able to say if those cells are malignant or benign.

However, let’s imagine we have a dataset of tons of cells, benign and malignat, the analysis of this dataset could lead to several observations. Differences in the characteristics between benign and malignant cells.

You can use the values given in this dataset in order to say if a new cell is benign or malignant.
You will have to clean the data, then select the algorithm that fits the best the problem and train your model, in order to make it understand the pattern of a benign and the pattern of a malignant cell. Once the model is trained, it can be used to predict the class of an unknown cell. This is Machine Learning.

Machine learning

Machine learning is the subfield of computer science that gives “computers the ability to learn without being explicitly programmed”

Arthur Samuel, American pioneer in the field of computer gaming and artificial intelligence.

Imagine you have a dataset of animals. And you want to build an algorithm able to recognize them. Your first task is to analyse the data you have. Do what we call Feature Extraction. For each picture you will create a vector with some variables :

  • Size of the eyes
  • Does it have a tail ?
  • Does il have wings ?
  • Tail size
  • …

Once you have all those elements, you will build a set of rules saying that an animal with medium sized eyes and a tell without any wings should be a cat.

In this case we explicitly programmed the algorithm to detect such element in our dataset and we may be working with hardcoded thresholds.
This method is a total failure. It is easy to understand that we will need a lot of rules in order to increase the precision of our algorithm, and this algorithm will not be easy to generalize to new data.

Machine Learning replace the hardcoded rules. Machine learning will use all the features we have from the picture and for each animal, it will learn the corresponding pattern. We do not give it the rules, just a way to learn them from the data.

Machine Learning is assimilable to a 4 years old child. It will start by understanding the data before starting to recognize animals.

Machine Learning has already an impact on our lifes. We can find a lot of exemples of Machine Learning applications.

  1. Netflix or Amazon recommendation system. It is an algorithm based on the shows you watch, items you buy, and then will recommend similar items.
  2. A lot of companies, use Machine Learning algorithms to predict if a customer will resign or not.
  3. We are starting to see applications in medicine.

Major Machine Learing Structure

  1. Recommendation System

    • It is build on what peoples likes, and it will predict what someone will like based on what he already liked and what people with the same “tast” like.

  2. Regression Estimation

    • The goal of regression is to predict continuous values from the data. For exemple, let us take the price of a house. A predicting algorithm will give us the price of an uknown house based on the characteristics of this house (size, #room, #floor …)

  3. Classification

    • The goal of classification is to predict the category of an item. Tell us if a cell is benign or malignant.

  4. Clustering

    • In a way Clustering is very similar to Classification in a way. The idea is to understand the structure of a data in order to being able to understand it. Find similar cases for exemple.

  5. Associations

    • Find relations between items. Associating items when they are co-occuring several times. It is used for exemple in supermarket. A studied showed that people who buy diaper, frequently buy beers, so they put the beers next to the diapers.

  6. Anomaly Detection

    • The idea is to go through a set of data and being able to recognize when a case isn’t a normal case. It is for exemple used in Card fraud detection. It will look through all the transactions made with a credit card, and try to see if all those transactions are correct or not.

  7. Sequence Mining

    • The principles of sequence mining is to being able to predict what will happened, based on what happened. It is very useful for NLP (Natural Language Processing), understand if a word is a Noun or a verb based on the previous words.

There is of course a lot of other types of Machine Learning Structure, as Reinforcement learning for exemple.

We have seen a lot of exemples of Machine Learning structure, however a question may be raised … what is the difference between Machine Learning and Artificial Intelligence ?

Artificial Intelligence is corresponding to all the method used to make a computer more “intelligent”. It is human-based and there is a lot of fields in it as Computer Vision for exemple.
Machine Learning is a part of Artificial Intelligence. It is the statistical part. It makes a computer learn how to solve a problem, based on collected data.
If we want to go “deeper”, we can also speak about Deep Learning. Deep Learning is a part of Machine Learning.

Supervised Learning VS Unsupervised Learning

In machine Learning we considere two types of learning methods :

  1. Supervised Learning
  2. Unsupervised Learning

But what is the difference between those methods.

Supervised Learning

First of all, what does supervised means ?

Control the activity of one, or multiple, person.

In our case we will not supervise someone, but “something”. The idea is to supervised our algoorithm.
How can we do that ?

We already saw that in machine learning we give a huge amount of data to a computer to make it learn rules. The idea is to train the model with some knowledge. The data is labelised, in order to make the model understand it.

If we take the exemple of cells classifier. We want an algorithm to tell us if a cell is benign or malignant.

When we will give the model the data, the data will have several element

  • The cell size
  • The cell perimeter
  • The cell area
  • …

All those elements are called features. Each cells will be a combination of all those features.
Moreover, we will give each cell a label. In our case the label is rather a cell is benign and malignant.
The idea is to make our model learn, the combination of features for benign and malignant cells. Like that, when it will face a new combination of features, it will be able to classify the unknown cell.

Supervised learning concerns to type of Machine Learning Structure :

  • Classification
  • Regression

In both cases we will assign a label to a combination of features and make the model learn which label correspond to which kind of combination.

Unsupervised Learning

Unsupervised learning is like it sounds, the opposite of supervised learning. We will not teach our model, but we will let him learn alone from the data.

The main difference is that we are still giving to our algorithm the features, but not giving the label anymore. Our data is unlabeled.

It is based on the fact that the algorithm will learn paterns not that humans will not see or understand.

The main structure used in unsupervised learning are

  • Clustering
  • Dimension Reduction
  • Density Estimation
  • Market Basket Analysis (Association)

The principle of Clustering is close to classification. We want to group similar items together in order to create clusters, however we do not know those groups.
