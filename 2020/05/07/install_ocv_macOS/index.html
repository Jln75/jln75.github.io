Install OpenCV On OSX
Created 2020-05-07| Updated 2020-05-08|CourseInstallationMacOS
Post View:

Our goal is to setup OpenCV on a MacOS computer. The idea is to be able to compile C++ code using OpenCV.
In this tutorial, we will use CMake to compile our programs.

Installing XCode

The first step is to install XCode. To do so, you just have to use the Apple Store and get the last version of XCode.

If the last version of XCode released on the Apple Store is not compatible with your OS, you should find the right XCode version to use using this table.

Once you know which version of Xcode you need, go to Apple Developer website, Login if you have an apple developer account, or just create one, and then you just have to find the needed version of XCode.

Once XCode is installed, just open it and accept xcode-build license.

Install OpenCV

Once XCode is installed, we can move to OpenCV Installation.

Install the requirements

A simple way to install packages on OSX is to use homebrew.
So the fist step will be to install homebrew on our device

Once we have installed Homebrew, we add it to the path

Now we can start to install our requirements.
OpenCV can be used in python or C++. We will use it for C++ but we can still install the requirements for Python.

We will install python 3, to be able to use OpenCV in python. It will also require CMAKE and Qt.

To configure the build directory we will use CMake. I recommend to download CMake directly from their website.

Opencv with C++

To install OpenCV and make it work in C++ needs more steps.

First we need to clone the OpenCV repositories. I chose to clone them in my /opt/ folder :

bash

We will clone them directly in ocv-git. We will also check if we are using the last stable version.

bash

Now we have all sources needed (opencv_extra is used to access tests).

Now it is time to use cmake. Open your CMake software. It should look like mine.

Select the path to the source code of opencv/ main repository. In my case it is in /opt/ocv-git/opencv/.

For “Where to build the libraries” choose the same path but add build/ at the end (/opt/ocv-git/opencv/build/).

Once it is done, click on configure button. It will ask you if the software can create the build folder, just say yes.

Once you configured it once, you will have access to the main parameters. Here you can choose what you want to build then.

In our case we took

  • OPENCV_ENABLE_NONFREE
  • WITH_QT
  • WITH_TBB
  • INSTALL_C_EXAMPLES
  • ENABLE_FAST_MATH

Once you have selected everything just click once again on configure and then on generate.

We will not need CMake anymore. Feel free to close the application.

Let us go back to our shell. We go to our build folder opencv in order to build it.
